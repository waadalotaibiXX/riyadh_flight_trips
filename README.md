Riyadh Airport Flight Activity Analysis (RUH)

Project Overview
This project analyzes flight activity data related to King Khalid International Airport (RUH) in Riyadh.  
The goal is to explore flight movements and understand operational patterns using real-world flight data.

The project focuses on structured data analysis and time-based exploration to support data-driven insights into airport flight activity.

Objectives
- Analyze flight activity associated with Riyadh Airport (RUH)
- Explore airlines, aircraft types, and flight movements
- Examine scheduled flight times and hourly distribution
- Clean and structure raw flight data for analysis

Dataset
The project uses cleaned and structured flight data related to Riyadh airport.

Key dataset attributes include:
- Flight number and airline information
- Aircraft model details
- Arrival flight records
- Origin and destination airports (ICAO / IATA)
- Scheduled time (UTC and local)
- Hourly flight activity

The dataset is provided in both CSV and Parquet formats for flexibility.

Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- Parquet & CSV data formats
- GitHub for version control and documentation
